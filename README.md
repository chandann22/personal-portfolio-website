## ✨ Features

- **Responsive Design** — Works seamlessly on mobile, tablet, and desktop
- **Animated Hero Section** — Typewriter-style animation cycling through roles
- **About Section** — Profile info, skills with progress bars
- **Resume Section** — Education, certifications, and achievements
- **Projects Showcase** — Cards with zoom-hover effect linking to GitHub repos
- **Contact Section** — Address, email, phone, and social links
- **Resume Download** — Direct PDF download from the site
- **Smooth Scroll & Animations** — Powered by AOS, ScrollAx, and Owl Carousel
- **Counter Stats** — Animated numbers for projects, achievements, and DSA problems

## 🚀 Getting Started

### View Locally

No build tools required — just open `index.html` in your browser.

```bash
git clone https://github.com/chandann22/personal-portfolio-website.git
cd personal-portfolio-website
open index.html   # or double-click the file
```

### Editing SCSS

If you want to modify styles via SCSS, compile using:

```bash
sass scss/style.scss css/style.css --watch
```
